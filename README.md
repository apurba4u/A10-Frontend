# Fable (Frontend)
A Next.js/React frontend for the Fable ebook platform.

**Live URL:** [https://fable-tau.vercel.app](https://fable-tau.vercel.app)

## Features
- Home, Browse, Ebook Detail pages.
- Authentication (Better Auth) with Google OAuth support.
- Role-based Dashboard (User, Writer, Admin).
- Stripe Checkout integration for purchases and writer verification.
- Dark Mode toggle.

## Tech Stack
- Next.js (App Router), Tailwind CSS, Shadcn UI, Axios, @tanstack/react-query, React Hook Form + Zod, Framer Motion, Recharts, next-themes, Better Auth, Stripe.

## Setup
1. `npm install`
2. Configure `.env` based on `.env.example`
3. `npm run dev`
